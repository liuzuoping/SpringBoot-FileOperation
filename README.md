# SpringBoot-FileOperation

## 介绍
SpringBoot集成MybatisPlus、Swagger、Mysql、Logback实现本地与OSS两种方式的文件上传、下载与预览功能的后端实现，前端只需调用接口即可，可本地postman自测
springboot-file
